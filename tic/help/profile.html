Das Profile

Im Profile kann jeder Spieler folgende Einstellungen festlegen:

  • Spielerinformationen (Punkte, Scantyp, SVs, SBs, Exen, Devensiveinheiten, ...)
  • Bevorzugtes Zeitformat
  • Neues Passwort vergeben
  • Eingabe von einer Telefonnummer und ICQ_Nummer (freiwillig!)
  • Urlaubsmodus ein- bzw. ausschalten
